

Switch to the right node version:  

```
nvm use 
```

Install Dependencies: 

```
yarn install 
```


Build Contracts:  


```
npx hardhat compile 
```

Test: 

```
npx hardhat test
```

Generate Test Coverage Report: 

```
npx hardhat coverage 
```

Generate ABI:

```
npx hardhat abi-gen
```
